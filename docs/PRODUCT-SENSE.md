# Product Sense

## Product Vision

개발자 포트폴리오 + 블로그 + CMS — 기술적 역량과 학습 여정을 보여주는 개인 브랜딩 플랫폼

### 핵심 가치
1. **전문성 전달**: 프로젝트, 기술 스택, 학습 기록을 체계적으로 전시
2. **콘텐츠 자율 관리**: Admin CMS로 코드 수정 없이 콘텐츠 CRUD
3. **접근성**: 누구나 쉽게 둘러볼 수 있는 깔끔한 퍼블릭 UI

## User Mental Model

### 방문자 (Public)
- **메인 페이지**: 포트폴리오 개요, 핵심 프로젝트
- **Projects**: 프로젝트 상세 (기술 스택, 기능, 스크린샷)
- **Insights**: 블로그/인사이트 글 읽기, 태그 필터, 아카이브
- **Study**: 학습 기록 열람
- **Contact**: 문의 폼 제출

### 관리자 (Admin)
- **Dashboard**: 콘텐츠 통계, 접속 로그 요약
- **콘텐츠 관리**: Insights/Studies/Features CRUD
- **사용자 관리**: 역할(Admin/Editor/Viewer) 기반 접근
- **설정**: 사이트 이름, SEO, 유지보수 모드

## Prioritization Framework

| 우선순위 | 기준 |
|----------|------|
| P0 | 보안 취약점, 데이터 손실, 사이트 다운 |
| P1 | 핵심 사용자 흐름 장애 (글 작성/조회 불가) |
| P2 | UX 개선, 새 피처 (SEO 최적화, 검색 기능) |
| P3 | 코드 품질, 리팩토링, 문서화 |
